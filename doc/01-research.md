# 01. 자료조사

## 국회도서관 식단표 시스템 분석

### 데이터 소스
- **URL**: 국회도서관 홈페이지 → 식단표 PDF
- **형식**: 주간식단표 PDF (매주 업데이트)
- **구조**: 도서관식당 & 박물관식당, 월~일 메뉴

### 기술적 과제
1. PDF 파싱 필요 (pymupdf 사용)
2. 한글 텍스트 처리
3. 주기적 자동 수집 필요
4. 모바일 접근성 확보

### 경쟁 서비스 분석
- 국회도서관 공식 홈페이지: PDF 다운로드만 가능
- 별도 모바일 앱 없음
- 실시간 알림 기능 없음

### 기술 선택
- **정적 사이트**: GitHub Pages (무료, 안정적)
- **자동화**: GitHub Actions (무료 플랜 충분)
- **프론트엔드**: Vanilla JS (라이브러리 의존성 최소화)
- **PWA**: 오프라인 지원, 홈화면 추가
