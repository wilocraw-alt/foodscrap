# 02. 계획수립

## 프로젝트 목표
국회도서관 주간식단표를 모바일 친화적 웹앱으로 제공

## 아키텍처 설계

### v1 (초기 프로토타입)
```
Python Flask 서버
  ↓ 실시간 크롤링
국회도서관 → PDF → pymupdf → SQLite
```

### v2 (프로덕션)
```
GitHub Actions (매일 09:00 KST)
  ↓ 자동 수집
data/menu.json → GitHub Pages → 브라우저
```

## 기능 명세

### 핵심 기능
1. 주간식단표 표시 (7일)
2. 도서관식당 & 박물관식당 구분
3. 모바일 반응형 레이아웃
4. 자동 이모지 매핑
5. 오프라인 캐싱 (PWA)

### 확장 기능
1. 식단 요청 게시판 (GitHub Issues)
2. 재수집 버튼
3. 오늘 날짜 표시
4. Last Updated 표시

## 기술 스택 결정

| 구분 | 선택 | 이유 |
|------|------|------|
| 프론트엔드 | Vanilla JS | 가벼움, 라이브러리 의존성 없음 |
| 스타일 | CSS3 | 별도 프레임워크 불필요 |
| 배포 | GitHub Pages | 무료, 자동 배포 |
| 자동화 | GitHub Actions | 무료 플랜 충분 |
| 데이터 | JSON | 파싱 간단, 브라우저 네이티브 지원 |
| 캐싱 | localStorage + SW | 6시간 캐시 + 오프라인 |
