# 🍽️ 국회도서관 식단표 - 사용자 가이드

## 서비스 개요
국회도서관 주간식단표를 모바일에서 편리하게 확인할 수 있는 웹앱입니다.

## 접속 방법
🔗 **웹사이트**: https://wilocraw-alt.github.io/foodscrap/

### 홈화면에 추가 (PWA)
1. 브라우저에서 위 URL 접속
2. Chrome: 우측 상단 ⋮ → "홈 화면에 추가"
3. Safari: 공유 버튼 → "홈 화면에 추가"
4. 앱 아이콘으로 빠르게 접근 가능

## 주요 기능

### 1. 주간식단표 확인
- **도서관식당**: 점심 메뉴
- **박물관식당**: 점심 메뉴
- **날짜 탭**: 월~일 탭 클릭하여 해당 날짜 메뉴 확인
- **오늘 표시**: 현재 요일에 빨간 "Today" 라벨

### 2. 메뉴 이모지
메뉴 항목에 자동으로 이모지가 표시됩니다:
- 🍖 고기류 (불고기, 돈까스, 갈비 등)
- 🍲 국물류 (국, 탕, 찌개)
- 🍚 밥류
- 🍜 면류 (국수, 라면, 우동 등)
- 🥗 샐러드
- 🐟 생선류
- 🥬 김치류
- 🍤 튀김류

### 3. 식단 요청
- **요청하기 버튼**: 새 식당이나 메뉴를 요청할 수 있습니다
- **최근 요청**: 다른 사람들이 요청한 내용을 확인할 수 있습니다
- 요청은 GitHub Issues로 관리됩니다

### 4. 재수집
- **🔄 재수집 버튼**: 최신 식단 데이터를 다시 불러옵니다
- 캐시된 데이터가 오래된 경우 사용하세요

### 5. 자동 업데이트
- 매일 오전 9시 (KST)에 최신 식단이 자동으로 수집됩니다
- 별도로 새로고침할 필요가 없습니다

## 자주 묻는 질문 (FAQ)

### Q: 메뉴가 "영업을 하지 않습니다"로 표시돼요
A: 해당 날짜에 식당이 영업하지 않는 것입니다. 주말이나 공휴일에 발생할 수 있습니다.

### Q: 메뉴가 표시되지 않아요
A: "🔄 재수집" 버튼을 눌러보세요. 그래도 안 되면 국회도서관에서 식단표를 아직 게시하지 않은 것입니다.

### Q: 식단 요청은 어떻게 하나요?
A: "💡 식단 요청하기" 버튼을 클릭하면 GitHub Issues 페이지로 이동합니다. 제목과 내용을 입력하고 "Submit new issue"를 클릭하세요.

### Q: 오프라인에서도 사용할 수 있나요?
A: 네, 한 번 접속하면 캐시된 데이터로 오프라인에서도 확인 가능합니다. 다만 최신 데이터를 보려면 인터넷 연결이 필요합니다.
